Proyek & Karya Siswa

Showcase hasil karya dan proyek terbaik dari siswa TEJEKATWO dalam bidang teknologi

Website E-Commerce

Platform jual beli online dengan fitur lengkap menggunakan PHP dan MySQL. Dilengkapi dengan sistem manajemen produk, keranjang belanja, dan payment gateway.

Proyek PKL
PHP MySQL

Konfigurasi Server

Setup dan konfigurasi web server, database server, dan file server untuk jaringan lokal sekolah. Menggunakan Linux Ubuntu Server dan Cisco router.

Tugas Praktik
Linux Cisco

Aplikasi Mobile Monitoring

Aplikasi Android untuk monitoring jaringan dan sistem komputer berbasis IoT. Memungkinkan administrator memantau status jaringan secara real-time.

Proyek P5
Android IoT

Desain Jaringan LAN

Perancangan dan implementasi jaringan LAN untuk gedung perkantoran multi-lantai. Menggunakan topologi star dengan VLAN segmentation untuk keamanan.

Tugas Praktik
VLAN Cisco Packet Tracer

Sistem Keamanan Jaringan

Implementasi firewall, IDS/IPS, dan VPN untuk mengamankan infrastruktur jaringan perusahaan dari ancaman siber. Proyek hasil PKL di instansi pemerintah.

Proyek PKL
Firewall VPN

Smart Garden IoT

Sistem penyiraman tanaman otomatis berbasis IoT menggunakan Arduino dan sensor kelembaban tanah. Dapat dimonitor dan dikontrol via smartphone.

Proyek P5
Arduino IoT
